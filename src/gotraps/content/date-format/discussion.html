The time.Time.Format and time.Parse functions expect a layout as first parameter. This layout doesn't consist in initials (Y, M, D...). It is actually the desired formatted string of the reference time Mon Jan 2 15:04:05 -0700 MST 2006 (equals to Unix time 1136239445), and it is not the value of the input date.

Although useful, this convention is a difficulty for beginners.
